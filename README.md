# Focus Timer

跨平台计时专注应用，支持 Windows、macOS、Linux、Android。采用 Flutter 开发，原生性能，多端体验一致。

## 核心功能

### 计时器系统
- **单核工作法**：自动计算到最近整点或半点，保证最少专注时长
- **番茄工作法**：专注 25min → 短休息 5min → 循环 → 长休息 15min（每 4 轮）
- 支持循环模式、自动开始下一轮
- **全方位通知系统**：专注或休息结束时触发铃声提醒、Windows 通知中心 Toast 以及应用内弹窗提醒

### 任务管理
- 多清单支持，自定义清单创建、编辑、删除
- **跨清单拖拽**：长按任务拖拽到侧边栏其他清单，快速移动任务
- 重复任务：每天、每周、每月、每年循环
- 截止日期提醒，逾期任务红色高亮
- "我的一天"特殊视图

### 数据同步 (Cloud Sync)
- **多端同步协议**：基于 LWW (Last-Write-Wins) 的增量同步算法
- **软删除支持**：确保删除操作在所有设备间正确传播
- **自动触发**：任务变动及专注会话结束后自动同步至云端
- **反馈机制**：UI 实时显示同步状态及错误提示

## 最近更新 (v1.0.5)

### 🚀 新特性
- **增强型专注交互**：
    - **继续专注选项**：当一轮专注结束时，现在支持直接点击“继续专注”。无论是在 Windows 通知中心、底部状态栏还是主计时面板，均可一键开启下一轮专注。
    - **快捷计时控制**：正在计时期间，点击左下角的倒计时按钮可快速切换计时面板的显示/隐藏，实现沉浸式任务处理。
- **自动化云同步**：
    - **全自动初始化**：账号登录或注册成功后，系统会自动触发全量同步，确保新设备数据立即就位。
    - **智能数据流转**：专注结束后自动触发后台同步，并实时刷新任务列表，确保多端任务状态高度同步。

### 🛠️ 优化与修复
- **通知系统精准化**：
    - **阶段逻辑纠正**：修复了通知标题与按钮在专注结束瞬间由于状态提前更新而导致的显示错误（如专注结束显示“休息结束”）。现在通知能准确反映刚刚完成的阶段。
    - **动作响应增强**：为通知中心按钮动作添加了诊断日志，并优化了热重载后的监听器挂载逻辑。
- **UI 细节打磨**：
    - **底部栏焕新**：重构了左下角状态按钮。放弃了厚重的系统原生按钮，采用轻量级自定义交互组件，解决了多按钮并列时的视觉拥挤感，适配深色模式。
    - **日历视图优化**：整体包裹滚动视图，彻底解决了小窗模式下的溢出（Overflow）报错；同时提升了网格密度，视觉排版更紧凑。
- **稳定性增强**：
    - **状态固化修复**：修复了应用重启后仍停留再“已完成”状态的 Bug，现在重启会自动重置为待机状态。
    - **编译错误修复**：解决了计时器模块因缺少 Provider 引用导致的 Windows 平台编译失败。

## 项目结构

```
lib/
├── main.dart                     # 应用入口
├── app.dart                     # 主框架
├── core/                        # 核心配置
│   ├── theme/                   # 主题配置
│   ├── services/                # 业务服务 (通知、音频等)
│   ├── utils/                  # 工具函数
│   └── providers/              # 全局 Provider
├── data/                        # 数据层
│   ├── database/               # SQLite 数据库 (支持迁移与软删除)
│   └── sync/                   # 同步服务 (协议实现)
└── features/                   # 功能模块
    ├── timer/                  # 计时器 (支持单核/番茄钟)
    ├── tasks/                  # 任务管理 (清单/重复/拖拽)
    ├── sidebar/               # 侧边栏 (右键菜单/清单编辑)
    ├── calendar/               # 日历视图 (专注记录概览)
    └── settings/               # 设置页面 (同步配置/调试信息)
```
